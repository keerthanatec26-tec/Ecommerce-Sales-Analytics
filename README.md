 ⚡E-Commerce Sales & Customer Analytics Dashboard
--------------------------------------------------------------------------------------------------------------------------------------------------------------------
This project presents an end-to-end data analysis of an e-commerce dataset, focusing on revenue performance, customer behavior, and delivery efficiency using Power BI.

The goal of this project is to transform raw data into meaningful insights that support data-driven decision-making.

🔧 Tools & Technologies
------------------------
Microsoft Excel – Data cleaning and preprocessing
Power BI – Data modeling, DAX, and dashboard development

📁 Dataset
------------
Brazilian E-Commerce Public Dataset (Olist)
Contains data on orders, customers, products, payments, and reviews

📌 Key Features
----------------
Interactive dashboard with multiple pages
KPI tracking (Revenue, Orders, Average Order Value, Customer Rating)
Customer segmentation and geographic analysis
Product and category performance analysis
Delivery performance and customer satisfaction insights

⚡ Key Insights
----------------
Credit cards are the most commonly used payment method
A small group of customers contributes a large portion of total revenue
Delivery delays have a negative impact on customer review scores
Sales performance varies across product categories and time periods
